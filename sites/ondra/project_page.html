The descriptions of all the showcased projects are presented below.

Sprite animation

In the first project, a simple sprite animation is displayed in a canvas window. The animation consists of multiple states, which can be selected using a dropdown menu. All the animations are derived from a single sprite sheet, and based on the selected state, the appropriate line of frames is extracted and displayed. The state and frame details are managed through an object that defines the number of frames and the position of each animation, ensuring smooth transitions between states.

Parallax backgrounds

Second showcase demonstrates a parallax background effect using five distinct layers, each moving at a different speed. The varying speeds of the layers create a convincing 3D effect, adding depth to the scene. Each layer is represented by an object with a unique image and horizontal speed, and to ensure smooth transitions, two moving images are used for each layer. Additionally, a slider allows you to adjust the speed of the backgrounds, giving you control over the parallax effect.

NPC movement animation

This project demonstrates NPC (non-player character) movement using instances of an Enemy object, all derived from a single image. The animation technique is similar to the one used in the sprite animation showcase. NPC positions are updated using a random function from the Math library, and movement is restricted to prevent NPCs from going beyond the borders. While all NPCs are identical in this example, the setup can be modified to incorporate different enemy types and movement patterns in the future. A slider at the bottom controls the number of NPCs, ranging from 0 to 30. Adjusting the slider deletes old instances, creates new ones, and recalculates their starting positions, offering a dynamic and interactive simulation.

Playing animations on trigger

This project showcases triggered animations with sound effects. When you click, an explosion animation is displayed in the canvas window, accompanied by a sound effect. Each click generates a new instance of the explosion object, with the starting angle randomized to ensure that every explosion appears unique. A dropdown menu allows you to choose between two triggers: click or mouse move.
When click is selected, each click triggers both the explosion animation in the canvas window and the sound effect. If mouse move is selected, new explosion instances are created as you move the mouse within the canvas area, but without any sound effects. The mousemove option is available only on PC
